# EKO-TEKNOPARK

EKO-TEKNOPARK projesi için hazırlanmış web tabanlı yönetim paneli arayüzüdür.

## Sayfalar

Projede şu sayfalar bulunmaktadır:

- Ana Sayfa
- Otomasyon ve Kontrol
- Raporlama
- Ayarlar

## Kullanılan Teknolojiler

- React
- TypeScript
- Vite
- Tailwind CSS

## Kurulum ve Çalıştırma

```bash
pnpm install
pnpm dev
```

Daha sonra tarayıcıdan şu adres açılır:
http://localhost:5173
